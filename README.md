A modern, responsive personal portfolio website built with **HTML, CSS, and JavaScript**.
This portfolio showcases my background, technical skills, and featured software projects in a clean, animated, and user-friendly design.

The website uses a light visual theme with pink and cream aspects, animated gradients, smooth scroll effects, a typing animation, responsive layouts, and interactive project sections.

## 🌐 Live Website

**View the portfolio here:**
  `https://compsciwro.github.io/rolandesolomons-portfolio-website/`

## ✨ Features

* Fully responsive design for desktop, tablet, and mobile
* Dark mode inspired interface
* Animated gradient backgrounds
* Glassmorphism-style project and content cards
* Typing text effect in the hero section
* Smooth scroll reveal animations
* Mobile-friendly navigation menu
* Project showcase section
* Contact section with form validation
* Built with plain HTML, CSS, and JavaScript — no frameworks required

## 🛠️ Technologies Used

* **HTML5** — page structure and content
* **CSS3** — styling, layout, responsiveness, and animations
* **JavaScript** — interactivity, typing effect, menu behavior, scroll reveal, and form validation
* **Font Awesome** — icons
* **Google Fonts** — typography

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:

```bash
git clone https://github.com/your-username/rolandesolomons-portfolio-website.git
```

2. Open the project folder:

```bash
cd rolandesolomons-portfolio-website
```

3. Open `index.html` in your browser.

No installation or build step is required.

## 🌍 Hosting with GitHub Pages

This portfolio can be hosted for free using GitHub Pages.

1. Push the project files to a GitHub repository.
2. Make sure `index.html` is in the root folder of the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/root` folder.
6. Save the settings.
7. GitHub will generate a live website link.

The site will be available at:

```text
https://your-username.github.io/rolandesolomons-portfolio-website/
```

## 🔧 Customisation

To personalise the portfolio, update the following:

* Replace placeholder GitHub, LinkedIn, and email links
* Add real project screenshots or demo images
* Update project descriptions and repository links
* Add a downloadable CV or resume
* Update education details
* Edit the rotating titles in the `script.js` file
* Connect the contact form to a service such as Formspree if email submission is needed

## 📬 Contact Form Note

The contact form currently includes front-end validation, but it does not send emails by default.
To enable email submissions, the form can be connected to a service such as Formspree, EmailJS, or a custom backend.
