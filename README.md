# Rolande Solomons — Portfolio Website

A modern, animated personal portfolio built with **plain HTML, CSS and JavaScript** —
no frameworks, no build step. Dark mode, glassmorphism cards, animated gradients,
typing effect, scroll reveals and a fully responsive layout.

## 📁 Files

| File | What it does |
|------|--------------|
| `index.html` | Page structure & all content |
| `style.css`  | All styling, animations & responsive rules |
| `script.js`  | Typing effect, menu, scroll reveals, contact form |

## ✏️ What you need to personalise

Search the project for `TODO` and `your-` to find every spot to edit. The main ones:

1. **Links** — replace every `https://github.com/your-username`, `https://linkedin.com/in/your-handle`
   and `your-email@example.com` with your real GitHub, LinkedIn and email.
   (They appear in the hero, projects, contact section and footer.)
2. **Project cards** (`index.html`, Projects section) — update the GitHub/demo links,
   and add real screenshot/demo URLs. Duplicate the placeholder card for new projects.
3. **Resume / CV** — add your `cv.pdf` to the repo and link it in the Contact section
   (look for `TODO: Link your CV/resume PDF`).
4. **Education** — update the university name and graduation year.
5. **Rotating titles** — edit the `phrases` array near the top of `script.js`.
6. **Contact form** — it validates but does not send email yet. To make it work,
   connect a free service like [Formspree](https://formspree.io) (instructions are in `script.js`).

## 🚀 Host it free on GitHub Pages

1. Push these files to your GitHub repository (the `index.html` must be in the repo root).
2. Go to **Settings → Pages**.
3. Under *Build and deployment*, set **Source** to `Deploy from a branch`.
4. Choose your branch (e.g. `main`) and folder `/ (root)`, then **Save**.
5. Wait ~1 minute — your site will be live at
   `https://your-username.github.io/rolandesolomons-portfolio-website/`.

## 🛠 Run locally

Just open `index.html` in any browser. (For the fonts/icons to load you need an
internet connection, since they come from CDNs.)
